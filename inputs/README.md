# Inputs — raw signal

Raw material only. Files here are captured, dated, and never rewritten — synthesis happens in `../hypotheses/` and `../synthesis/`, which link back here.

## Conventions

- Filename: `YYYY-MM-DD-slug.md`
- Start each file with a short header: source, date, participants/context.
- End each file with a `## Signal` section: 1–5 bullets of what this input suggests, and which hypothesis it touches (e.g., `→ H2`).

## Subdirectories

| Dir | Source | How it gets here |
|---|---|---|
| `calls/` | Fireflies transcripts / call notes | No Fireflies MCP connector exists yet — paste transcripts or export summaries manually, or drop exported files here and ask Claude to process them |
| `slack/` | Notable Slack threads | Via Slack connector once authorized (claude.ai connector settings), or paste |
| `linear/` | Roadmap/product signal | Via Linear connector once authorized, or paste |
| `customers/` | Asks, objections, win/loss reasons | Captured from calls, emails, anywhere |
| `notes/` | Kurt's raw thinking | Write freely; Claude files the signal |
| `research/` | External research: competitors, market, DOSS public positioning | Claude web research, analyst pieces, press |
