# Synthesis

Dated synthesis passes (`YYYY-MM-DD-synthesis.md`), run roughly weekly or whenever meaningful new signal has accumulated in `../inputs/`.

Each pass answers four questions:

1. **What new signal arrived?** (list the new input files)
2. **Which hypotheses moved?** (status changes, new evidence for/against)
3. **What changed in STRATEGY.md?** (or explicitly: nothing yet, and why)
4. **What are the top open questions now?**

When a synthesis produces something worth presenting to a wider group, turn it into a deck in `../deck/` (see [deck/README.md](../deck/README.md)).
